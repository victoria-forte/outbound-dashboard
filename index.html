Connect Your Google Sheet

Power this dashboard with a shared Google Sheet so the whole team sees the same data. Create a sheet with the right structure, publish it, and paste the URL below.

1

Create the Sheet

Create a Google Sheet with 3 tabs named Weekly, Monthly, Quarterly. Each tab should have columns: Name, Role, Region, Calls, Meetings, Connects, Opps, PBR, Closed.

2

Add a Targets Row

Optionally add a row with Name = TARGET in each tab to set per-metric targets. If omitted, defaults are used.

3

Publish to Web

Go to File → Share → Publish to web and publish the entire document as a web page. Then paste the Sheet URL above.
